# 🌐 Portfólio Pessoal - Kauã Borges

Este repositório contém o código-fonte do meu portfólio pessoal. O projeto foi desenvolvido com foco em um design limpo, moderno e com navegação estruturada, servindo como meu cartão de visitas para apresentar minhas habilidades como Desenvolvedor Full Stack.

---

## 📸 Sobre o Layout

O projeto utiliza um design estilo **Split-Screen (Tela Dividida)** com um menu lateral fixo e uma área de conteúdo dinâmica:
* **Menu Lateral:** Fundo escuro contratando com textos em tom *Azure*, botões com cantos arredondados e efeito de transição (*hover*).
* **Área de Conteúdo:** Cards organizados com bordas laterais destacadas para facilitar a leitura das seções (Sobre Mim, Habilidades, Formação, Projetos e Contato).

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica das seções e blocos de conteúdo.
* **CSS3:** Estilização avançada utilizando posicionamento fixo (`position: fixed`), manipulação de box model (`padding`/`margin`), efeitos dinâmicos de interação (`:hover`) e paleta de cores customizada em RGB.

---

## 📂 Estrutura de Arquivos

* `index.html` - Estrutura de marcação e linkagem das seções do portfólio.
* `style.css` - Identidade visual, tipografia (Arial/Helvetica), cores e regras de layout do site.
